## Wstęp

Repozytorium zawiera program będący implementacją generatora i algorytmu rozwiązującego problem plecakowy w języku Java. Celem jego napisania było zapoznanie się ze składnią języka.

## Klasy
### Item
Klasa definiująca obiekt przedmiotu. Jego cechami są wartość oraz waga.

### Result
Klasa definiująca rozwiązanie problemu. Jego cechami jest ilość przedmiotów, suma wag i suma wartości przedmiotów. Ponadto przechowuje listę ID przedmiotów.

### Problem
Klasa definiująca cechy problemu (rodzaje przedmiotów, ziarno generatora oraz przedział liczb w którym generowane będą cechy przedmiotów) oraz funkcje: generującą problem oraz rozwiązującą go.

### Main

Funkcja główna programu, wywoływane jest wygenerowanie problemu oraz jego rozwiązanie, ponadto deklarowana jest tam zmienna pojemności plecaka. Użytkownik może określić liczbę rodzajów przedmiotów oraz ziarno.
